🐾 Guess the Dog Breed

I extended the original quote machine into a fun and interactive dog breed guessing game using the Dog CEO API. The app displays a random image of a dog, and the user has to guess its breed by typing into an input box. When the user clicks "Guess Breed," the app provides instant feedback — letting them know if they got it right or showing the correct breed if they didn’t. I added image loading, user input, real-time validation, and a reveal feature to make it engaging and educational!
